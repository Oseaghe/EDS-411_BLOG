## Project Overview

EDS-411_BLOG is a simple and responsive blogging platform built with HTML, CSS, and JavaScript. The aim of the project is to demonstrate key concepts in web development, including front-end design, interactivity, and styling.

## Features

- Responsive blog layout
- Easy-to-navigate interface
- Basic post creation and display functionality (expand as features are added)
- Modern styling using CSS

- **JavaScript** (25.4%)

## Getting Started

To use or contribute to this project, follow these steps:

1. **Clone the repository**
   ```bash
   git clone https://github.com/Oseaghe/EDS-411_BLOG.git
   ```
2. **Open the project in your preferred code editor.**
3. **Open `index.html` in your browser** to view the blog.
